# hsConfig
Config Lib for HS


# About

Create secure config files with RSA
Loosely based on JWT

# Properties

- typ   Type
- alg   Algorithm
- iss   Issuer
- sub   Subject
- aud   Audience
- exp   Expires
- nbf   Not Before
- iat   Issued At
- cid   Config ID
- nam   Service Name
- zon   Zone
- sid   Service ID
